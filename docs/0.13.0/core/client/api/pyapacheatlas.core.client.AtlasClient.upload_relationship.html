pyapacheatlas.core.client.AtlasClient.upload_relationship¶

AtlasClient.upload_relationship(relationship)¶

Upload a AtlasRelationship json. Should take the form of the following:

{
    "typeName": "hive_table_columns",
    "attributes": {},
    "guid": -100,
    "end1": {
        "guid": assignments["-1"]
    },
    "end2": {
        "guid": assignments["-5"]
        }
}
Parameters

relationship (dict) – The relationship you want to upload.

Returns

The results of your relationship upload.

Return type

dict
