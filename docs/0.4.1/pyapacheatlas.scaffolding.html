pyapacheatlas.scaffolding.column_lineage module¶

pyapacheatlas.scaffolding.column_lineage.column_lineage_scaffold(datasource, use_column_mapping=False, column_attributes=[], table_attributes=[], table_column_relationship_attributes=[], column_lineage_attributes=[], table_process_attributes=[], column_lineage_process_attributes=[], table_process_column_lineage_relationship_attributes=[])¶

Create a base set of type definitions that adhere to the Hive Bridge style of Atlas Column Lineage.

Parameters

  • datasource (str) – The name of the data source. Acts as a prefix for all other type defs.

  • use_column_mapping (bool) – If True, add the columnMapping attribute to the table process.

  • optional column_attributes (list(dict),) – Attribute Defs to add to the column entity type.

  • optional table_attributes (list(dict),) – Attribute Defs to add to the table entity type.

  • optional table_column_relationship_attributes (list(dict),) – Attribute Defs to add to the table column relationship type.

  • optional column_lineage_attributes (list(dict),) – Attribute Defs to add to the column lineage process entity type.

  • optional table_process_attributes (list(dict),) – Attribute Defs to add to the table process entity type.

  • optional column_lineage_process_attributes (list(dict),) – Attribute Defs to add to the column lineage process entity type.

  • optional table_process_column_lineage_relationship_attributes (list(dict),) – Attribute Defs to add to the table process column lineage relationship type.
